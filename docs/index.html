Welcome to the Star Wars Analysis Project

The Star Wars analysis project is a collaborative effort by Jordan James, Shaun Massenburg, Jaxon Abele, and Zachery Schleger. For this project, we work to analyze trends in the scripts from Star Wars Episodes 1-6. Throughout this project there are several insteresting visualizations that show the analysis work we have been doing.
